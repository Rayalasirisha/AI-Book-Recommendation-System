#  AI Book Recommendation System

An AI-powered Book Recommendation System that provides personalized book suggestions using **Machine Learning**, **Collaborative Filtering (SVD)**, and an interactive **Streamlit** web application.

##  Project Overview

Choosing the right book from thousands of available titles can be difficult. This project solves that problem by recommending books based on users' previous ratings and reading preferences.

Multiple recommendation algorithms were implemented, evaluated, and compared. The **Tuned Singular Value Decomposition (SVD)** model achieved the best performance and was selected as the final recommendation engine.

#  Machine Learning Models

The following recommendation models were evaluated:
- Popularity-Based Recommendation
- Content-Based Filtering
- Baseline Only
- KNN Basic
- NMF
- Singular Value Decomposition (SVD)
- Neural Collaborative Filtering (NCF)
- DeepFM
- Wide & Deep

#  Final Model
Tuned Singular Value Decomposition (SVD)

### Evaluation Metrics
| Metrics| Score |
| RMSE | 1.84 |
| MAE | 1.44 |

The Tuned SVD model delivered the best recommendation accuracy among all tested models.
